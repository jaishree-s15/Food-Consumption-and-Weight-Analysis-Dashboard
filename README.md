Food Consumption and Weight Analysis Dashboard using Power BI
Project Overview

This interactive Power BI dashboard explores the relationship between food consumption patterns and weight categories. The analysis examines various cuisines, meal types, calorie intake, and nutritional factors to identify trends that influence body weight and dietary preferences.

Key Features
Analysis of multiple food categories including Indian, Italian, Persian, Thai, and Turkish cuisines.
Comparison of food preferences across different weight groups.
Calorie consumption analysis and daily calorie tracking.
Nutritional value comparison by weight category.
Meal-wise analysis including breakfast, soup, and waffle consumption.
Interactive slicers for filtering data by weight groups.
KPI visualization for monitoring average calories consumed per day.
Insights Generated
Popular food choices among different weight categories.
Impact of calorie intake on weight distribution.
Nutritional patterns associated with various weight groups.
Meal preferences and their relationship with body weight.
Comparative analysis of international cuisine consumption trends.
Tools Used
Power BI Desktop
Power Query
DAX Measures
Data Modeling
Interactive Visualizations
Business Value

This dashboard helps understand dietary behaviors and weight-related trends, enabling data-driven insights for nutrition analysis, health monitoring, and lifestyle research.
